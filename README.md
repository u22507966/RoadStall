# RoadStallSales Frontend

This is the frontend for my RoadStallSales app.

I made it to help manage the daily running of a roadside stall in one place.

I built it with Angular and used it to handle the day to day side of the system like stock, sales, users, profile settings, and notifications.

## What it does

- Login and register users
- Manage products and stock
- Capture daily sales
- Run stock take flow
- Manage users and roles
- Send stock request notifications

## Run it locally

```bash
npm install
ng serve
```

Then open `http://localhost:4200/`

## Notes

This repo is the frontend only.

The backend is separate, so some features need the API running to work properly.
